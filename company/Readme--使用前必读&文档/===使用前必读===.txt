【本项目信息】

1.开发环境：macOS 10.15.1  +  Python 3.8.1 +  Django 3.0.6

2.适用系统：win7及以上系统、macOS10.X及以上系统、Unix内核的Ubuntu、Linux、CentOS等支持Python3.8的系统

3.适用的Python版本： Python 3.8.1（项目开发时最新的版本，不支持低版本python，请安装py3.8，而3.8以上的版本无法测试，请自行选择）


【使用必读】

1.项目存放在百度云盘，为防止屏蔽，会不定期修改下载链接，请及时下载到本地或保存至自己的云盘中；

2.从百度云盘下载项目时，如未开通会员or网速不好时，会出现丢文件的情况（尤其是小文件），会导致项目启动过程中出现找不到文件的报错，请重新下载项目文件，替换整个项目即可（无需删除新建的虚拟环境，无需删除已加载的依赖包）；

3.无论是用于学习，还是用于搭建个人网站，请务必先在本地使用终端启动项目（自己的win或macOS系统的电脑上启动），验证项目的完好性，并熟悉一下项目的启动过程；首次启动，请勿使用Pycharm！！！
